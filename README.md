# Проект "Игры разума"

Убедитесь, что у вас установлен uv.

Проект включает в себя 7 консольных игр.
Для установки используйте следующие команды:

1. Cборка проекта

        make build

2. Установка пакетов

        make package-install

3. Запуск необходимой игры

        # Выводит приветствие
        brain-game

        # Игра: "Проверка на чётность" 
        brain-even

        # Игра: "Калькулятор"
        brain-calc

        # Игра: "НОД"
        brain-gcd

        # Игра "Арифметическая прогрессия"
        brain-progression

        # Игра "Простое ли число?"
        brain-prime
